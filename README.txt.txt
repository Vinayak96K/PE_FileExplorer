Portable Executable File Explorer (PE File Explorer)
This project gives information about various headers of an .exe or .dll file.
It provides Character User Interface (CUI).

The following list contains code files and information about contents
PE_FileExplorer.cpp - this file contains the entrypoint function (main()).
pch.h - It contains delcaration of prototypes of global functions and classes.
HelperFile.cpp - this file contains defination of global function, constructors and member functions of all classes.
